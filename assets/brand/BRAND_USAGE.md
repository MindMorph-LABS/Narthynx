# Narthynx Brand Assets

Use these files in the OSS repo instead of uploading the full brand-board screenshot.

## Recommended usage

- `narthynx-logo-lockup-dark.svg` — README hero/banner.
- `narthynx-icon.svg` — GitHub social preview, docs, website favicon base.
- `narthynx-icon-cyan.png` — fallback for platforms that do not render SVG.
- `narthynx-cli-card.svg` — CLI/demo card in docs.

## Suggested repo path

```txt
assets/
  brand/
    narthynx-icon.svg
    narthynx-icon-white.svg
    narthynx-icon-cyan.png
    narthynx-logo-lockup.svg
    narthynx-logo-lockup-dark.svg
    narthynx-cli-card.svg
```
